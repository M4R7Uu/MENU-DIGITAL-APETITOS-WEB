# 🥘 Apetitos - Menú Digital Interactivo

Proyecto de menú digital optimizado para dispositivos móviles, diseñado para el restaurante **Apetitos** en La Apartada, Córdoba. 

Desarrollado por **<IF>CODE**.

## 🚀 Características
- **Diseño Mobile-First:** Optimizado para una visualización rápida y clara en celulares.
- **Navegación Sticky:** Categorías fijas para facilitar el acceso a Desayunos, Almuerzos y Especiales.
- **Destaque Visual:** Resalte de platos estrella mediante CSS avanzado (Glow & Gradients) sin sobrecargar el peso de la página.
- **Integración con WhatsApp:** Botón flotante con mensaje automatizado para pedidos directos.
- **Branding Personalizado:** Créditos de desarrollador con animaciones sutiles.

## 🛠️ Tecnologías Utilizadas
- **HTML5** (Semántico)
- **CSS3** (Flexbox, Grid, Animations, Variables)
- **JavaScript** (Manipulación de DOM para pestañas interactivas)
- **Google Fonts** (Arvo & Outfit)

## 📱 Vista Previa
El sitio utiliza una paleta de colores cálidos (`#fdd55c`, `#3e2723`) que refleja la identidad del local físico.
